# MCP
    Um padrao open source para conectar aplicacoes AI em sistemas externos.
    Usando  NCPS, aplicacoes Claude e CHatGPT pode conectar a data sources (arquivos locais
    base de dados remotas ou locais), ferramentas, workflows, permitindo a elas acesso a informacoes
    para realização de tarefas.

    MCP é como uma porta USB-C para apps AI, USB-C prove uma interface padronizada para c
    onectar uma variedade de dispositivos.


    Exemplo

        Chat Interfacae
            - Claude Desktop, LibreChat
        IDEs
            - Claude Code, Goose

        [Chat, IDE] -> MCP <- [Data and file systens, Dev Tools, Productivity Tools]
